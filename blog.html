Question1 : Discuss the scope of var, let, and const

Ans : Var,let and Const are Keyword That used to declare variable, Var is function and global scope That means you can access the variable value from anywhere but the value will be undefined,On the Other side both let and Const are block scope means you can't access variable those has been declared with let and const outside of the Block.

Question2 :Tell us the use cases of null and undefined

Ans : Undefined Is a built in value in javascript that automatically assign to a Variable if the variable is declared but no value has assigned; The value Undefined can be found if You call a function that don't return anything in that case the result will be undefined.On the other side Null is used to indicate that the Property has no meaning full value so the person on the other side can understand by simply looking at the code.

Question3 :What do you mean by REST API?

Ans : REST stands for "Representational State Transfer," and a REST API (Application Programming Interface) is a set of rules and conventions for building and interacting with web services over HTTP. It is a widely used architectural style for designing networked applications, particularly web services.
